## Response Pattern Summary

### Pattern 1: Direct Answer + Next Steps
Used for: Colors, Sizes, Features
```
[Emoji] [Question Type] is available in/includes:
[Specific Information]

[Follow-up suggestion]
```

### Pattern 2: Availability Affirmation + Options
Used for: Stock checks, availability questions
```
✅ Yes! [Product] is available in these styles:
• [Style 1]
• [Style 2]

[Call-to-action]
```

### Pattern 3: Detailed Information
Used for: Comprehensive questions
```
📦 [Product] Information:

**[Product Name]**
[Description]
🎨 Colors: [list]
📏 Sizes: [list]

[Next steps]
```

### Pattern 4: Help & Options
Used for: Help requests
```
📦 Available Product Categories:
[list]

❓ I can answer questions about:
[capabilities]

[Example questions]
```

## Tips for Best Results

1. **Be Specific:** Mention product names for more accurate responses
2. **Ask Naturally:** The bot understands casual, conversational questions
3. **Use Keywords:** Words like "color", "size", "material" help route to right handler
4. **Follow Suggestions:** The bot suggests logical follow-up questions
5. **Ask for Help:** If unsure, start with "Help" or "What can you do?"
